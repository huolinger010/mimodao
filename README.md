# 🏛️ MiMoDAO

**AI-Powered DAO Governance Intelligence**

Analyze proposals, generate governance documents, and explore top DAOs — 100% free, no API key required.

## ✨ Features

- **🔍 Proposal Analyzer** — Select a DAO, view active proposals, get instant multi-dimensional analysis (complexity, financial impact, security, governance, technical, urgency, sentiment) with risk assessment and voting recommendations
- **✍️ AI Writer** — Describe your idea in plain language, generate a professional DAO proposal with Summary, Motivation, Specification, Budget, Timeline, Metrics, and Risk Mitigations
- **🌐 DAO Explorer** — Browse 30+ top DAOs on Snapshot with member counts and follower stats

## 🚀 Live Demo

**https://huolinger010.github.io/mimodao/**

## 🛠️ Tech Stack

- Single HTML file, zero dependencies
- [Snapshot.org](https://snapshot.org) GraphQL API (free, no key)
- Rule-based intelligent analysis engine
- Responsive dark/light theme
- Modern glassmorphism UI with smooth animations

## 📋 Supported DAOs

ENS · Uniswap · Aave · Lido · Arbitrum · Optimism · Gitcoin · Safe · Curve · MakerDAO · dYdX · Sushi · Compound · Nouns · Morpho + 30+ more via Explorer
